HTML

  • HTML is made up of elemets which work together to provide content and the basic framework for all pages on the web
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Semantic HTML elements describe its content and the role it plays on the webpage in the name

CSS

  • CSS is a programming language used by developers to define how webpage content is presented to users
  • The three ways to style a webpage using CSS include inline CSS, internal CSS style sheet, and external CSS style sheet
  • Declarations contain two components: the CSS property we want to apply and the value of the property
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element

Git

  • git status: checks what branch we are currently working on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript is a high-level dynamic programming language that allows users to interact with websites
  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • Conditional statements and loops can control the order in which code is executed, if at all
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over
